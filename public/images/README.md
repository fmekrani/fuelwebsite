# Image Assets

Place your app screenshots in this directory with the following names:

## Required Screenshots

- **home-screen.jpg** - Main dashboard/home screen
- **feed-screen.jpg** - Social feed with friends' activities
- **leaderboards-screen.jpg** - Competition and rankings
- **calories-screen.jpg** - Calorie and macro tracking interface
- **analytics-screen.jpg** - Analytics and insights dashboard
- **run-tracking-screen.jpg** - GPS run tracking view
- **workout-plans-screen.jpg** - Workout plans and exercises

## Additional Assets

- **favicon.svg** - App icon/favicon (already included)
- **og-image.jpg** - Open Graph image for social sharing (1200x630px recommended)

## Image Specifications

- **Format**: JPG or PNG
- **Width**: 750px recommended for phone screenshots
- **Optimization**: Compress images for web to improve load times
- **Aspect Ratio**: Match iPhone dimensions (e.g., 9:19.5 for iPhone 14)

## Temporary Placeholders

Until you add your actual screenshots, the site will show placeholder backgrounds. The layout and functionality will work perfectly - just swap in your images when ready!

## Quick Export from Xcode Simulator

1. Run your app in the iOS Simulator
2. Navigate to each screen you want to capture
3. Press `Cmd + S` to save screenshot
4. Optimize and rename according to the list above
5. Copy to this directory

---

**Tip**: Use an image optimization tool like TinyPNG or ImageOptim to reduce file sizes without losing quality.
